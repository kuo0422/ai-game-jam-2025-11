# 像素風格平台系統說明

## 🎨 完成項目

### 1. **平台類型標註（levelData.js）**
所有平台都已經標註了類型，共有 4 種類型：

| 類型 | 說明 | 用途 | 視覺風格 |
|------|------|------|----------|
| `floor` | 地板 | 可站立的地面 | 深棕色石頭，帶磚塊紋理 |
| `platform` | 平台 | 浮空平台 | 青藍色石頭，光滑質感 |
| `wall` | 牆壁 | 垂直牆面 | 灰褐色磚牆，錯縫排列 |
| `ceiling` | 天花板 | 洞穴頂部 | 深灰色岩石，粗糙質感 |

### 2. **像素風格渲染器（platformRenderer.js）**
創建了專門的渲染系統，使用程式繪製像素風格的材質，無需圖片資源。

## 📝 使用方法

### **在 levelData.js 中標註平台類型**
```javascript
platforms: [
    // 地板
    { id: 1, x: 0, y: 2350, width: 1000, height: 50, type: 'floor' },
    
    // 平台
    { id: 2, x: 200, y: 2250, width: 200, height: 30, type: 'platform' },
    
    // 牆壁
    { id: 3, x: 0, y: 2000, width: 50, height: 400, type: 'wall' },
    
    // 天花板
    { id: 4, x: 0, y: 2000, width: 300, height: 50, type: 'ceiling' }
]
```

### **自動渲染**
系統會根據 `type` 屬性自動選擇對應的繪製樣式。無需額外配置！

## 🔧 自定義修改

### **修改顏色**
在 `platformRenderer.js` 中找到對應的繪製函數，修改 `ctx.fillStyle` 的顏色值：

```javascript
// 例如：修改地板的主體顏色
ctx.fillStyle = '#4a3520'; // 改成你想要的顏色
```

### **修改磚塊大小**
修改 `tileSize`、`brickWidth`、`brickHeight` 等變數：

```javascript
const tileSize = 16; // 改成 32 會變成更大的磚塊
```

### **添加新類型**
在 `PlatformRenderer` 類中添加新的繪製方法：

```javascript
static drawMetal(ctx, x, y, width, height) {
    // 你的金屬材質繪製代碼
}
```

然後在 `draw()` 方法的 switch 中添加新的 case：

```javascript
case 'metal':
    this.drawMetal(ctx, platform.x, platform.y, platform.width, platform.height);
    break;
```

## 🎨 設計理念

- **地板**：溫暖的棕色，像古老的石地板
- **平台**：冷色調的藍色，營造浮空感和魔法氛圍
- **牆壁**：中性的灰褐色，堅固可靠
- **天花板**：深色灰色，營造洞穴的壓抑感

每種材質都使用高光和陰影來創造深度感，符合像素藝術的經典風格！
