# 🗺️ 遊戲地圖指南

## 地圖概覽

這是一個 **5000 x 2400 像素** 的超大無縫地圖，玩家可以自由探索，沒有房間切換或載入畫面。

## 📍 區域分布

```
        [隱藏區]          [Boss通道]─────────┐
          ⬆️                  ⬆️              │
          │                  │              │
     [爬升區]      [探索區-能力球]       [Boss房]
          ⬆️                  ⬆️              ⬆️
          │                  │              │
     [起始區] ──→ [主通道-存檔] ──→ [深坑] ──→ [挑戰區]
```

### 🎮 各區域詳細說明

#### 1️⃣ 起始區 (左下角)
- **位置**: x: 0-1000, y: 2000-2400
- **特色**: 玩家出生點，簡單的平台配置
- **敵人**: 2 個巡邏兵（新手友善）
- **路線**: 可往右走進主通道，或往上爬升

#### 2️⃣ 主通道 (中下)
- **位置**: x: 1000-2500, y: 2000-2400
- **特色**: 長走廊，有存檔點 💾
- **敵人**: 4 個混合敵人（巡邏+追擊）
- **重要**: 存檔點在 x: 1750 附近

#### 3️⃣ 深坑 (右下)
- **位置**: x: 2500-3500, y: 2000-2400
- **特色**: 有一個大坑洞，坑內有平台
- **敵人**: 3 個敵人
- **挑戰**: 需要精準跳躍

#### 4️⃣ 爬升區 (左中)
- **位置**: x: 0-1000, y: 1400-2000
- **特色**: 垂直爬升的平台階梯
- **敵人**: 3 個守衛
- **通往**: 隱藏區

#### 5️⃣ 探索區 (中間)
- **位置**: x: 1000-3000, y: 1400-2000
- **特色**: 複雜的平台佈局，能力球在此 ✨
- **敵人**: 5 個強力敵人
- **重要**: 能力球在 x: 1750, y: 1500

#### 6️⃣ 挑戰區 (右中)
- **位置**: x: 3000-4000, y: 1400-2000
- **特色**: 難度較高的平台跳躍
- **敵人**: 2 個追擊型敵人
- **獎勵**: 高難度挑戰區

#### 7️⃣ 隱藏區 (左上)
- **位置**: x: 0-1000, y: 800-1400
- **特色**: 需要二段跳才能完全探索
- **敵人**: 無（和平區域）
- **秘密**: 最高處可能有隱藏獎勵

#### 8️⃣ Boss 通道 (中上)
- **位置**: x: 1000-3000, y: 800-1400
- **特色**: 通往 Boss 房的階梯
- **敵人**: 3 個精英守衛
- **Boss 門**: 需要二段跳能力才能開啟 🚪

#### 9️⃣ Boss 房 (右上)
- **位置**: x: 3000-5000, y: 800-1400
- **特色**: 超大競技場
- **敵人**: 3 個強力守衛 + Boss
- **挑戰**: 最終戰

## 🎯 推薦探索路線

### 🔰 新手路線（無二段跳）
```
起始區 → 主通道 → 存檔 → 探索區 → 收集能力球
```

### ⭐ 完整路線（有二段跳）
```
取得能力球 → 隱藏區 → Boss 通道 → 開啟 Boss 門 → Boss 房
```

## 💡 重要提示

### 存檔點 💾
- **位置**: 主通道 (x: 1750, y: 2130)
- **功能**: 
  - 靠近時顯示「按 E 休息」
  - 可恢復血量（待實作）
  - 作為復活點（待實作）

### Boss 門 🚪
- **位置**: Boss 通道 (x: 2520, y: 1050)
- **條件**: 需要二段跳能力
- **狀態**: 
  - 🔒 紅色 = 鎖住
  - 🔓 金色 = 可開啟

### 能力球 ✨
- **位置**: 探索區 (x: 1750, y: 1500)
- **能力**: 二段跳
- **重要性**: 必須收集才能進入 Boss 房

## 🎨 視覺標記

遊戲會自動顯示：
- 右上角顯示當前區域名稱
- 靠近存檔點時顯示互動提示
- 靠近 Boss 門時顯示開啟條件

## 🗺️ 地圖大小參考

- **總大小**: 5000 x 2400 像素
- **螢幕可見**: 1280 x 720 像素
- **探索範圍**: 約需要左右移動 4 個螢幕寬度
- **垂直範圍**: 約需要上下移動 3 個螢幕高度

## 🎮 操作提示

- **移動**: A/D 或 方向鍵
- **跳躍**: 空白鍵 或 W
- **攻擊**: J 或 K
- **互動**: E（靠近存檔點或門時）
